Project: Self-hosted Cloud Lab (AWS + Open Source)

✅ Cloud Lab Setup (Self-hosted, AWS + Open Source) – Built self-hosted lab using AWS Free Tier and local virtualization for training and testing.

Objective:

Deploy a cost-effective, scalable cloud lab environment using AWS Free Tier and open-source virtualization tools to support internal testing, PoC validations, and trainer-led sessions.

Summary:

  • Provisioned 30+ EC2 instances with customized system specs tailored for software simulations, hands-on embedded tools, and cloud infrastructure training.
  • Utilized Ubuntu, Docker, and other open-source technologies to replicate real-world deployment scenarios.
  • Configured IAM, VPC, and Security Groups to ensure isolation and secure access across training environments.
  • Integrated local virtualization (VirtualBox) for hybrid labs and offline access when needed.
  • Provided trainers with consistent terminal access and role-based resource provisioning using separate AWS Organization accounts.

Key Highlights:

  • Designed fully modular cloud lab for testing, demonstrations, and continuous learning
  • Optimized for AWS Free Tier to reduce cost without compromising functionality
  • Enabled multi-user training access with granular IAM policies
Skills Demonstrated: AWS EC2 | IAM | VPC | Ubuntu | Docker | VirtualBox | Open Source Tools | Training Lab Design | Infrastructure Simulation

Lab Status: Completed
