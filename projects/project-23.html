Project: SharePoint File Server Implementation

✅ SharePoint File Server Implementation– Migrated on-premises file structure to cloud-based SharePoint with access control and sync.

Objective:

Implement a centralized and secure file server using SharePoint Online to replace local shared drives with modern collaboration and permission control.

Summary:

  • Designed departmental document libraries in SharePoint to replicate traditional file server structure.
  • Configured sync using OneDrive client to enable offline access for users across departments.
  • Implemented granular permissions and access controls via Microsoft 365 groups and SharePoint roles.
  • Trained staff to use SharePoint for file sharing, version control, and secure access from anywhere.

Key Highlights:

  • Replaced legacy file server with SharePoint Online across departments
  • Ensured department-level isolation and secure sharing using M365 access controls
  • Enabled auto-sync and offline access with OneDrive integration
Skills Demonstrated: SharePoint Online | Microsoft 365 | File Server Migration | Access Control | OneDrive Sync | Cloud Storage Management

Status: Completed – In Production
